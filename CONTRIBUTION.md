# Contributing to Nuviojs

Thank you for your interest in contributing to Nuviojs.

Nuviojs is fully open-source and welcomes contributions from developers, designers, and ecommerce experts.

## How to Contribute

1. Fork the repository and clone your fork
2. Create a new branch off main
3. Make your changes
4. Run lint and tests (if applicable)
5. Push and open a pull request with a clear description

## Local Setup

- Use npm for package management
- Prisma is used for DB migrations and seeding
- Tailwind and shadcn/ui are used for styling
- Linting is configured with eslint and prettier

## Install dependencies
npm install
## Run the development server
npm run dev
